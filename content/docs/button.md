# GButton

> **Not yet wrapped in Gandalf.** Until `GButton` is implemented, use the shadcn/ui primitive directly:
>
> ```ts
> import { ... } from '@/components/ui/button'
> ```

A wrapper around shadcn/ui's `Button`. Supports multiple visual variants (`default`, `destructive`, `outline`, `ghost`, `link`) and sizes, with full CVA-driven variant definitions.
