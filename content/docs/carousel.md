# GCarousel

> **Not yet wrapped in Gandalf.** Until `GCarousel` is implemented, use the shadcn/ui primitive directly:
>
> ```ts
> import { ... } from '@/components/ui/carousel'
> ```

A wrapper around shadcn/ui's `Carousel`, built on top of the Embla Carousel library. Supports horizontal and vertical orientations, and optional autoplay.
