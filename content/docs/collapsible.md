# GCollapsible

> **Not yet wrapped in Gandalf.** Until `GCollapsible` is implemented, use the shadcn/ui primitive directly:
>
> ```ts
> import { ... } from '@/components/ui/collapsible'
> ```

A wrapper around shadcn/ui's `Collapsible`, built on top of Reka UI's `Collapsible` primitive. Simpler than `Accordion` — use when you need a single show/hide toggle.
