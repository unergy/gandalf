# GContextMenu

> **Not yet wrapped in Gandalf.** Until `GContextMenu` is implemented, use the shadcn/ui primitive directly:
>
> ```ts
> import { ... } from '@/components/ui/context-menu'
> ```

A wrapper around shadcn/ui's `ContextMenu`, built on top of Reka UI's `ContextMenu` primitive. Triggered by a secondary mouse click (right-click) on its target element.
