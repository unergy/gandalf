# GEmpty

> **Not yet wrapped in Gandalf.** Until `GEmpty` is implemented, use the shadcn/ui primitive directly:
>
> ```ts
> import { ... } from '@/components/ui/empty'
> ```

A custom Gandalf primitive for empty states. Displays a message and optional call-to-action when a list, table, or content area has no data to show.
