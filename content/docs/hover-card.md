# GHoverCard

> **Not yet wrapped in Gandalf.** Until `GHoverCard` is implemented, use the shadcn/ui primitive directly:
>
> ```ts
> import { ... } from '@/components/ui/hover-card'
> ```

A wrapper around shadcn/ui's `HoverCard`, built on top of Reka UI's `HoverCard` primitive. Use to show a rich preview on hover without blocking the user's flow.
