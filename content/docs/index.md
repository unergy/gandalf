# Gandalf UI

A component library built on top of [shadcn/ui](https://ui.shadcn.com/) with extended variants, colors, and behaviors designed for the Unergy design system.

## Components

| Component                  | Description                                              |
| -------------------------- | -------------------------------------------------------- |
| [Badge](/docs/badge)       | Versatile badge with colors, variants, shapes, and sizes |
| [Dropdown](/docs/dropdown) | Dropdown menu with a fully-customizable slot trigger     |
| [Tabs](/docs/tabs)         | Tabs with default, underline, and pills variants         |
