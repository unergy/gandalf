# GInputOtp

> **Not yet wrapped in Gandalf.** Until `GInputOtp` is implemented, use the shadcn/ui primitive directly:
>
> ```ts
> import { ... } from '@/components/ui/input-otp'
> ```

A wrapper around shadcn/ui's `InputOtp`, built on top of `vue-input-otp`. Provides individual character slots with automatic focus management — ideal for verification codes and PIN entry.
