# Installation

Gandalf UI requires **Nuxt 3**, **Tailwind CSS v4**, and **shadcn-nuxt**. Follow these steps to get it running from scratch.

---

## 1. Create a Nuxt project

```bash
bun create nuxt my-app
cd my-app
```

## 2. Install dependencies

```bash
bun add -D tailwindcss @tailwindcss/vite
bun add reka-ui class-variance-authority clsx tailwind-merge lucide-vue-next
```

## 3. Configure Tailwind

In `nuxt.config.ts`, register the Vite plugin:

```ts
import tailwindcss from '@tailwindcss/vite'

export default defineNuxtConfig({
  vite: {
    plugins: [tailwindcss()],
  },
})
```

Create `app/assets/css/tailwind.css` and import it:

```ts
// nuxt.config.ts
css: ['~/assets/css/tailwind.css'],
```

```css
/* tailwind.css */
@import 'tailwindcss';
```

## 4. Install shadcn-nuxt

```bash
bun add -D shadcn-nuxt
```

Add the module to `nuxt.config.ts`:

```ts
modules: ['shadcn-nuxt'],
shadcn: {
  prefix: '',
  componentDir: './app/components/ui',
},
```

Run the init command to scaffold the base theme:

```bash
bunx shadcn-nuxt@latest init
```

## 5. Copy the Gandalf layer

Clone or copy the `app/components/gandalf/` directory into your project. This folder contains the `base/` and `kit/` layers built on top of the `ui/` primitives.

```
app/components/
  ui/          ← generated by shadcn (do not edit)
  gandalf/
    base/      ← copy this
    kit/       ← copy this
```

## 6. Add the utility helper

Gandalf components use a shared `cn()` utility. Add it to `app/lib/utils.ts`:

```ts
import { clsx, type ClassValue } from 'clsx'
import { twMerge } from 'tailwind-merge'

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs))
}
```

## 7. Import and use components

Components are imported directly from their folder:

```vue
<template>
  <GBadge color="success">Active</GBadge>
  <GButton variant="default">Save</GButton>
</template>

<script setup lang="ts">
import { GBadge } from '@/components/gandalf/badge'
import { GButton } from '@/components/gandalf/button'
</script>
```

## Requirements

| Dependency | Version |
|---|---|
| Nuxt | `^3.13` |
| Tailwind CSS | `^4.0` |
| shadcn-nuxt | `^0.11` |
| Reka UI | `^2.0` |
| class-variance-authority | `^0.7` |
| lucide-vue-next | `^0.400` |
