# GMenubar

> **Not yet wrapped in Gandalf.** Until `GMenubar` is implemented, use the shadcn/ui primitive directly:
>
> ```ts
> import { ... } from '@/components/ui/menubar'
> ```

A wrapper around shadcn/ui's `Menubar`, built on top of Reka UI's `Menubar` primitive. Ideal for application-level navigation bars with multi-level menus.
