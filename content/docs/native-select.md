# GNativeSelect

> **Not yet wrapped in Gandalf.** Until `GNativeSelect` is implemented, use the shadcn/ui primitive directly:
>
> ```ts
> import { ... } from '@/components/ui/native-select'
> ```

A custom Gandalf primitive wrapping the native `<select>` element. Use when a fully accessible, no-JavaScript fallback is required or when rendering inside complex form contexts.
