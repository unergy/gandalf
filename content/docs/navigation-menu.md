# GNavigationMenu

> **Not yet wrapped in Gandalf.** Until `GNavigationMenu` is implemented, use the shadcn/ui primitive directly:
>
> ```ts
> import { ... } from '@/components/ui/navigation-menu'
> ```

A wrapper around shadcn/ui's `NavigationMenu`, built on top of Reka UI's `NavigationMenu` primitive. Use for top-level site navigation with optional flyout menus.
