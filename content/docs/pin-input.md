# GPinInput

> **Not yet wrapped in Gandalf.** Until `GPinInput` is implemented, use the shadcn/ui primitive directly:
>
> ```ts
> import { ... } from '@/components/ui/pin-input'
> ```

A wrapper around shadcn/ui's `PinInput`, built on top of Reka UI's `PinInput` primitive. Similar to `GInputOtp` but driven by Reka UI's primitive rather than a third-party OTP library.
