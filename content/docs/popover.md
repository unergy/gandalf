# GPopover

> **Not yet wrapped in Gandalf.** Until `GPopover` is implemented, use the shadcn/ui primitive directly:
>
> ```ts
> import { ... } from '@/components/ui/popover'
> ```

A wrapper around shadcn/ui's `Popover`, built on top of Reka UI's `Popover` primitive. Use to display rich content in a non-modal overlay — dates, filters, mini forms.
