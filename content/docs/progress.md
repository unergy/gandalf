# GProgress

> **Not yet wrapped in Gandalf.** Until `GProgress` is implemented, use the shadcn/ui primitive directly:
>
> ```ts
> import { ... } from '@/components/ui/progress'
> ```

A wrapper around shadcn/ui's `Progress`, built on top of Reka UI's `Progress` primitive. Displays a horizontal bar that reflects a numeric value between 0 and 100.
