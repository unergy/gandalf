# GRadioGroup

> **Not yet wrapped in Gandalf.** Until `GRadioGroup` is implemented, use the shadcn/ui primitive directly:
>
> ```ts
> import { ... } from '@/components/ui/radio-group'
> ```

A wrapper around shadcn/ui's `RadioGroup`, built on top of Reka UI's `RadioGroup` primitive. Supports `v-model` and keyboard navigation between options.
