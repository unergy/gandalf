# GRangeCalendar

> **Not yet wrapped in Gandalf.** Until `GRangeCalendar` is implemented, use the shadcn/ui primitive directly:
>
> ```ts
> import { ... } from '@/components/ui/range-calendar'
> ```

A wrapper around shadcn/ui's `RangeCalendar`, built on top of Reka UI's `RangeCalendar` primitive. Extends `GCalendar` with start and end date selection for date range inputs.
