# GScrollArea

> **Not yet wrapped in Gandalf.** Until `GScrollArea` is implemented, use the shadcn/ui primitive directly:
>
> ```ts
> import { ... } from '@/components/ui/scroll-area'
> ```

A wrapper around shadcn/ui's `ScrollArea`, built on top of Reka UI's `ScrollArea` primitive. Replaces the native scrollbar with a consistent, themed one across all browsers.
