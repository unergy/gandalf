# GSelect

> **Not yet wrapped in Gandalf.** Until `GSelect` is implemented, use the shadcn/ui primitive directly:
>
> ```ts
> import { ... } from '@/components/ui/select'
> ```

A wrapper around shadcn/ui's `Select`, built on top of Reka UI's `Select` primitive. Supports `v-model`, grouped options, disabled states, and custom trigger rendering.
