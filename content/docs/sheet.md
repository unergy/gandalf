# GSheet

> **Not yet wrapped in Gandalf.** Until `GSheet` is implemented, use the shadcn/ui primitive directly:
>
> ```ts
> import { ... } from '@/components/ui/sheet'
> ```

A wrapper around shadcn/ui's `Sheet`, built on top of Reka UI's `Dialog` primitive. Similar to `GDialog` but slides in from a screen edge (top, right, bottom, or left). Ideal for settings panels and detail drawers on desktop.
