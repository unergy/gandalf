# GSidebar

> **Not yet wrapped in Gandalf.** Until `GSidebar` is implemented, use the shadcn/ui primitive directly:
>
> ```ts
> import { ... } from '@/components/ui/sidebar'
> ```

A wrapper around shadcn/ui's `Sidebar`. Provides a structured, collapsible sidebar layout with header, content, and footer slots. Used as the primary navigation shell in the Unergy platform.
