# GSlider

> **Not yet wrapped in Gandalf.** Until `GSlider` is implemented, use the shadcn/ui primitive directly:
>
> ```ts
> import { ... } from '@/components/ui/slider'
> ```

A wrapper around shadcn/ui's `Slider`, built on top of Reka UI's `Slider` primitive. Supports `v-model`, min/max/step, vertical orientation, and multi-thumb range selection.
