# GSonner

> **Not yet wrapped in Gandalf.** Until `GSonner` is implemented, use the shadcn/ui primitive directly:
>
> ```ts
> import { ... } from '@/components/ui/sonner'
> ```

A wrapper around shadcn/ui's `Sonner`, built on top of the `vue-sonner` library. Place `<GSonner />` once in the app root to enable toast notifications anywhere via the `toast()` composable.
