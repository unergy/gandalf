# GSpinner

> **Not yet wrapped in Gandalf.** Until `GSpinner` is implemented, use the shadcn/ui primitive directly:
>
> ```ts
> import { ... } from '@/components/ui/spinner'
> ```

A custom Gandalf primitive for loading states. Displays an animated SVG spinner with configurable size and color. Use when an async operation is in progress and the result area is not yet available.
