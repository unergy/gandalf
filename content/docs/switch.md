# GSwitch

> **Not yet wrapped in Gandalf.** Until `GSwitch` is implemented, use the shadcn/ui primitive directly:
>
> ```ts
> import { ... } from '@/components/ui/switch'
> ```

A wrapper around shadcn/ui's `Switch`, built on top of Reka UI's `Switch` primitive. Supports `v-model:checked` and is the preferred alternative to a standalone checkbox for boolean settings.
