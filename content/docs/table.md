# GTable

> This component is under construction.

A wrapper around shadcn/ui's `Table` primitives, composable with [TanStack Table](https://tanstack.com/table). Provides `GTable`, `GTableHeader`, `GTableBody`, `GTableRow`, `GTableHead`, and `GTableCell` sub-components for building flexible data tables.
