# GTagsInput

> **Not yet wrapped in Gandalf.** Until `GTagsInput` is implemented, use the shadcn/ui primitive directly:
>
> ```ts
> import { ... } from '@/components/ui/tags-input'
> ```

A wrapper around shadcn/ui's `TagsInput`, built on top of Reka UI's `TagsInput` primitive. Lets users type and confirm multiple values, each displayed as a removable chip/tag.
