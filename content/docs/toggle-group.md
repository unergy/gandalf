# GToggleGroup

> **Not yet wrapped in Gandalf.** Until `GToggleGroup` is implemented, use the shadcn/ui primitive directly:
>
> ```ts
> import { ... } from '@/components/ui/toggle-group'
> ```

A wrapper around shadcn/ui's `ToggleGroup`, built on top of Reka UI's `ToggleGroup` primitive. Supports `single` and `multiple` selection types with `v-model`.
