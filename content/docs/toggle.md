# GToggle

> **Not yet wrapped in Gandalf.** Until `GToggle` is implemented, use the shadcn/ui primitive directly:
>
> ```ts
> import { ... } from '@/components/ui/toggle'
> ```

A wrapper around shadcn/ui's `Toggle`, built on top of Reka UI's `Toggle` primitive. Supports `v-model:pressed` and multiple visual variants. Use for toolbar actions like bold, italic, or view mode switches.
