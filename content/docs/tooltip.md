# GTooltip

> **Not yet wrapped in Gandalf.** Until `GTooltip` is implemented, use the shadcn/ui primitive directly:
>
> ```ts
> import { ... } from '@/components/ui/tooltip'
> ```

A wrapper around shadcn/ui's `Tooltip`, built on top of Reka UI's `Tooltip` primitive. Displays a short label when hovering over an interactive element. Requires `GTooltipProvider` at the app root.
